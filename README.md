# 📝 Blogs - Next.js Blog Platform

A modern blog platform built using Next.js with authentication, AI features, and database integration.

---

## 🚀 Tech Stack Used

- ⚡ Next.js (App Router)
- ⚛️ React.js
- 🎨 Tailwind CSS
- 🔐 Authentication (NextAuth)
- 🗄️ Supabase (Database & Backend)
- 🤖 AI Integration (GROQ_AI)
- 🧩 TypeScript
- ☁️ Vercel (Deployment)

---

## 🚀 Deployment Steps

### 🔹 Option 1: Vercel (Recommended)

1. Go to [https://vercel.com]
2. Import GitHub repository: `blogs`
3. Set root directory (if needed): `blogs`
4. Click **Deploy**

Vercel will automatically detect Next.js and handle build settings.
